## Dev Server Notes

- Always start with `npm run dev` from project root
- Dev server uses port 3000 by default; if 3000 is taken it bumps to 3001
- `NEXT_PUBLIC_APP_URL` in `.env.local` MUST match the actual running port — mismatch causes auth CORS errors
- Kill stale node processes with `taskkill //F //IM node.exe` before restarting if port issues occur
